# Visão Geral do Projeto LuzIA

O LuzIA é um sistema desenvolvido para apoiar organizações na realização de questionários de diagnóstico, especialmente voltados para a avaliação de aspectos psicossociais no ambiente de trabalho.

Por meio do sistema, é possível cadastrar organizações e seus setores, aplicar questionários de forma estruturada e coletar respostas dos participantes com garantia de anonimato, assegurando a privacidade das informações fornecidas.

Com base nas respostas coletadas, o LuzIA realiza a análise dos dados e gera diagnósticos, auxiliando empresas e gestores na compreensão de cenários, identificação de pontos de atenção e apoio à tomada de decisões.

O sistema foi projetado de forma organizada, permitindo que cada tipo de informação seja armazenado de maneira clara e independente, o que contribui para maior confiabilidade, segurança e facilidade de manutenção ao longo do tempo.
